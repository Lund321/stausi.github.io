# Perico Heist Script

<ResourceLinks link="https://shop.stausi.com/package/6531760" />

Take your FiveM server to the next level with the Perico Mansion Heist! This adrenaline-pumping experience is packed with thrilling gameplay, offering players the ultimate challenge as they attempt to infiltrate a heavily guarded mansion on Cayo Perico.

## Installation

<Steps>
### Install all resource dependencies
- [ox_lib](https://github.com/overextended/ox_lib)
- [ox_inventory](https://github.com/overextended/ox_inventory)
- [ox_target](https://github.com/overextended/ox_target)
- [minigames](https://shop.stausi.com/package/6526480)
- [teams](https://shop.stausi.com/package/6528185)

### Download the script from your keymaster

### Drag and drop the resource in your resources folder

### Add this ensure in your server.cfg
```bash
ensure st_perico_heist
```

### The perico heist script is ready to go!
</Steps>
