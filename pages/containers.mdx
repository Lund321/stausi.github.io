# Containers Script

<ResourceLinks link="https://shop.stausi.com/package/6525108" />

A container robbery script. Several containers spawns around the map, which the user can rob!

<Callout type="info">
This script require custom coding to integrate in your own script!
</Callout>

## Installation

<Steps>
### Install all resource dependencies
- [ox_lib](https://github.com/overextended/ox_lib)
- [ox_inventory](https://github.com/overextended/ox_inventory)
- [ox_target](https://github.com/overextended/ox_target)

### Download the script from your keymaster

### Drag and drop the resource in your resources folder

### Add this ensure in your server.cfg
```bash
ensure st_containers
```

### The containers script is ready to go!
</Steps>
